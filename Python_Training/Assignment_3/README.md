## Tasks Completed

### A1 - NumPy Basics

This module demonstrates fundamental NumPy operations.

**Features:**

* NumPy array creation
* Mean, maximum, minimum, and sum calculation
* Array addition and multiplication
* Matrix creation using NumPy

---

### A2 - Pandas DataFrame Operations

This module demonstrates how to work with tabular data using Pandas.

**Features:**

* Employee DataFrame creation
* Displaying records and summary statistics
* Filtering employees by department
* Bonus calculation using salary data

---

### A3 - Data Cleaning

This module demonstrates basic data cleaning techniques.

**Features:**

* Missing value detection
* Replacing missing age values with mean
* Replacing missing salary values with default values
* Preparing clean data for analysis

---

### A4 - GroupBy Analysis

This module demonstrates data analysis using Pandas GroupBy operations.

**Features:**

* Average salary by department
* Maximum salary by department
* Employee count by department
* Department-wise data aggregation

---

### A5 - Matplotlib Charts

This module demonstrates data visualization using Matplotlib.

**Charts:**

* Bar Chart
* Line Chart
* Histogram
* Scatter Plot

---

### A6 - Seaborn Visualizations

This module demonstrates statistical visualizations using Seaborn.

**Visualizations:**

* Barplot
* Boxplot
* Heatmap

---

### A7 - Student Performance Mini Project

This mini project combines Pandas, Matplotlib, and Seaborn to analyze student performance data.

**Features:**

* Student dataset loading from CSV
* Performance classification
* Study hours and marks analysis
* Data visualization using charts and plots
