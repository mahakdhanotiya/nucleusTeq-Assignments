# Python Training - Assignment 2

## Overview

This repository contains solutions for Assignment 2 of Python Training,
covering advanced Python concepts and practical implementations.

The assignment covers advanced Python concepts including:

- Exception Handling
- Iterators and Generators
- Functional Programming
- Regular Expressions
- Testing and Debugging
- Packaging
- Parallel Execution (Threading and Multiprocessing)

## Requirements

Python 3.10+

Install pytest:

```bash
pip install pytest
```

---

## Running Programs

Navigate to the required folder and execute:

```bash
python3 file_name.py
```

Example:

```bash
cd parallel_execution
python3 q1_threading_two_threads.py
```

---

## Running Tests

Execute all tests:

```bash
python3 -m pytest -v
```

Execute a specific test file:

```bash
python3 -m pytest test_prime_number.py -v
```
