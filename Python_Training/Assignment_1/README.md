## Topics Covered

### Python Basics

* Variables and Data Types
* Arithmetic Operations
* Conditional Statements
* Loops
* Functions
* Modules

### Data Structures

#### Lists

* Sum of elements
* Maximum element
* Sorting
* Removing duplicates
* Counting even and odd numbers
* Reversing a list

#### Tuples

* Accessing tuple elements
* Converting tuple to list
* Modifying tuple data

#### Sets

* Union
* Intersection
* Difference
* Removing duplicates

#### Dictionaries

* Accessing values
* Character frequency counting
* Merging dictionaries

### File Handling

* Creating files
* Writing data to files
* Reading files
* Counting lines, words, and characters
* Appending content
* Copying file content
* Searching words in files

### Object-Oriented Programming

* Classes and Objects
* Constructors
* Inheritance
* Encapsulation
* Polymorphism
